§2

A message on the wire

Every SiFT message is a 16-byte header, an AES-256-GCM ciphertext, and a 12-byte authentication tag. The header is bound into the tag as associated data; the nonce is the sequence number followed by six random bytes. Edit the payload, then flip tamper to corrupt one ciphertext byte and watch verification fail closed.
